# Downloads Directory

This directory contains downloadable files for the Tastyaana application.

## APK File

- **tastyaana.apk** - Android application package file
- **Current Status**: Placeholder file (replace with actual APK)

## How to Update

1. Build your Android application
2. Generate the APK file
3. Replace `tastyaana.apk` with the actual APK file
4. Ensure the file is accessible via `/downloads/tastyaana.apk`

## File Requirements

- File name must be `tastyaana.apk`
- File should be a valid Android APK
- File should be optimized for production use
- Consider code signing for security

## Testing

The download banner on the homepage will automatically use this file for downloads.
